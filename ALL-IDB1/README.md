This directory contains the training and model file, as well as the individual fold indicies, for the ALL-IDB1 dataset used in our studies. To run this code, open the train_ALL_IDB1.py file, and change the path on line 117 to the correct location of the ALL-IDB1 dataset on your computer.

Additionally, the code for generating heatmaps from a trained model file (.pth) is also included in the Spike_Activation_Map_ALL-IDB1.py file. To run this code, change the path on line 117 to the correct location of the ALL-IDB1 dataset on your computer, and the path to the desired model file on line 145.
