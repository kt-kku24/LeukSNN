# LeukSNN: A novel spiking neural network for efficient acute lymphoblastic leukemia diagnosis [Applied Sciences]

The code in this repository is for the ALL-IDB1, ALL-IDB2, and Acute Lymphoblastic Leukemia Image Dataset (ALLID) datasets. Included are individual train and model files for each dataset, the spike activation map code, and the training and validation splits used in our studies.

The datasets used in this study are available from the websites below, and are not stored here as they do not belong to us.

ALL-IDB1 and ALL-IDB2: https://scotti.di.unimi.it/all/

ALLID: https://www.kaggle.com/datasets/mehradaria/leukemia
