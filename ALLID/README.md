This directory contains the training and model file, as well as the individual fold indicies, for the ALLID dataset used in our studies. To run this code, open the train_ALLID.py file, and change the path on line 128 to the correct location of the ALLID dataset on your computer.

Additionally, the code for generating heatmaps from a trained model file (.pth) is also included in the Spike_Activation_Map_ALLID.py file. To run this code, change the path on line 121 to the correct location of the ALLID dataset on your computer, and the path to the desired model file on line 159.
